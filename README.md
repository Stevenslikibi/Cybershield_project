# CyberShield

Plateforme web de signalement des cyber-escroqueries locales à Pointe-Noire.

## Description

CyberShield permet aux citoyens de signaler des arnaques en ligne et aux enquêteurs de la brigade de cybercriminalité de traiter les dossiers.

## Technologies

- Frontend : HTML5, CSS3, JavaScript
- Backend : Node.js, Express
- Base de données : PostgreSQL

## Installation

```bash
cd backend
npm install
node server.js
```
